**轻舟渡万境，一智载千寻！**

## 📦 工具概览
轻舟 AI 工具箱是一套基于本地 llama.cpp 服务的 AI 辅助工具集，包含以下独立模块：

| 工具文件 | 主要功能 | 适用场景 |
|---------|---------|----------|
| chat_Ai.py | 有记忆的聊天助手 | 日常连续对话、写作辅助、需要记忆上下文的问答 |
| chat_Ai_no.py | 无记忆的聊天助手 | 单次问答、测试模型效果、翻译、代码生成 |
| ai_buddy.py | 带有性格配置的陪伴型助手 | 情感交流、创意启发、需要情绪价值的互动 |
| subtitle_translator.py | 双语字幕翻译工具 | 批量翻译 SRT/TXT 字幕，支持聊天式即时翻译 |
| format_converter.py | 文档/图片格式转换工具箱 | Markdown 转 Word/PDF、图片格式互转、批量重命名 |

**核心依赖**：所有聊天与翻译工具均需先启动 llama.cpp 服务（llama-server.exe）。

### ⚙️ 关键设置项说明
| 设置项 | 说明 |
|--------|------|
| 系统提示词 | 定义 AI 的角色和行为。ai_buddy.py 可通过性格预设快速填充 |
| 选择模型 | 从 llama-server 加载的模型中选取。带“(多模态)”标识支持图片识别 |

| 多模态模式 | 自动 / 仅CPU（省显存） / 禁用多模态 |
| 图片上传 | 仅多模态模型支持 |

## ❓ 常见问题 (FAQ)

### Q1：模型下拉框显示“请先启动 llama-server”？
**A**：请确保已通过 `启动AI.bat` 启动 llama.cpp 服务。

### Q2：上传图片后模型不识别？
**A**：
1. 选择带 **(多模态)** 标识的模型
2. 多模态模式不要设为“禁用”
3. 图片为正常格式：jpg/png/gif

### Q3：导出 Word 表格列宽被挤压？
**A**：Word 右键表格 → 自动调整 → 根据内容调整表格。

### Q4：转换 PDF 提示 xelatex not found？
**A**：PDF 需要 LaTeX 引擎。可先导出 HTML，再浏览器打印为 PDF。

### Q5：字幕翻译速度慢？
**A**：适当调高“最大并发数”，或开启 GPU 层数加速。

### Q6：文件保存在哪里？
- 聊天导出：`output/chat_exports/`
- 字幕翻译：`双语字幕输出/`
- 格式转换：`output/`

### Q7：界面显示异常？
**A**：清理浏览器缓存或更换浏览器，本工具基于 Gradio 6.x。
